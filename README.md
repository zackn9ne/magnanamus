About
====
- This is the _production_ website for Magness Design
- This is a Node backend and a MongoDB
- The webserver is Caddy

Install Frontend
===
1. `git clone`
2. `bower install`
3. `npm install`
4. `sudo gem install sass`
5. `gulp`
6.  curl `localhost:3000`
